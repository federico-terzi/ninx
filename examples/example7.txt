Let's see how blocks can be used to define custom objects:

$prototype = {
    $name = {Federico}

    @func test() {
        Your name is $name
    }

    @func setName($n) {
        $name = $n
    }
}

$obj1 = $prototype
$obj1.name = {Bob}

$obj2 = $prototype

$obj3 = $prototype
@obj3.setName {Jerry}

@obj1.test
@obj2.test
@obj3.test


Read: $obj1.name
